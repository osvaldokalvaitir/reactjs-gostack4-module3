## :wrench: Install and run

Open terminal:

```sh
# Clone this repo
git clone https://github.com/osvaldokalvaitir/reactjs-gostack4-module3

# Entry in folder
cd reactjs-gostack4-module3

# Install deps with npm or yarn
npm install | yarn

# Launch the app with npm or yarn
npm start | yarn start
```

Click to learn more about the tools used: [Reactotron](https://github.com/osvaldokalvaitir/awesome/blob/main/src/apps-inspectors/reactotron.md).
